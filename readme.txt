This plugin allows your store to generate invoices via Cargo Expreso service.


== Description ==

= Cargo Expreso - WooCommerce Invoicing =

This plugin will add a button on the wordpress dashboard page for editing orders. You can click a button to request the invoice, and another to send the invoice to your clients' email.

== Usage ==

To start using the "Cargo Expreso - WooCommerce Invoicing", first create an account at [CargoExpreso.com](https://cargoexpreso.com). They will provide you with your account "IdenEmpresa", "Token", "Private key" and "Public Key".


After you have your Cargo Expreso or sandbox account active:

1. Head to Woocommerce Settings and click on the Checkout tab.
2. On checkout options you should see the option "Cargo Expreso", click on it.
3. Enable the payment gateway byt checking the checkbox that reads "Enable this payment gateway".
4. Fill the form with your account information.  Dont forguet to check the "Enable Test Mode" box if your account is from the sandbox.
5. Click on save changes and you should be ready to start generating invoices with Cargo Expreso.

== Installation ==

Installing "Cargo Expreso  - WooCommerce Invoicing" can be done using the following steps:
	
1. Download the plugin.
1. Upload the ZIP file through the "Plugins > Add New > Upload" screen in your WordPress dashboard.
1. Activate the plugin through the 'Plugins' menu in WordPress
